# Required Images

Please add the following images to the `public` folder:

1. **logo.jpeg** - The site logo/company logo
   - Location: `/public/logo.jpeg`
   - Used in: Header (displayed as 64x64px rounded)

2. **m-reda.jpg** - Portrait of Dr. M. Reda Sholkami
   - Location: `/public/m-reda.jpg`
   - Used in: Homepage Section 1

## Image Recommendations

- **logo.jpeg**: Should be square or near-square, minimum 200x200px
- **m-reda.jpg**: Professional portrait photo, recommended minimum 600x800px
